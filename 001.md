# 2.基本语法
## 2.1 标题
一个#代表一级标题，两个#代表二级标题，最多有六级标题。
## 2.2 引用
一个>代表一级引用，两个>代表二级引用。
## 2.3 列表
有序列表：可以用数字
无序列表：可以用+-*
嵌套：空四格缩进
todolist：
- [x] a
- [ ] b
- [ ] c
## 2.4 表格
|左对齐|居中对齐|右对齐|
|:-----|:-----:|-----:|
|a|b|c|
## 2.5 段落
- 换行：空两格之后回车或者空一行
- 分割线：三个*
- 字体
  |字体|代码|
  |:--:|:--:|
  |*斜体*|* *|c
  |==高亮==|== ==|
  |**粗体**|** **|
  |***斜粗体***|*** ***|
  |~~删除~~|~~ ~~|
  |<u>下划线</u>|`<u> </u>`|
## 2.6 代码
``` 
print('Hello world!')
print('bye-bye')
```
